## Notes

- **Recall@k** uses binary Hit@k: 1 if any gold document appears in top-k, else 0. This is the standard RAG retrieval metric.
- **MRR** (Mean Reciprocal Rank): average of 1/rank of the first relevant document across all queries.
- **nDCG@k** uses binary relevance (1 if gold, 0 otherwise) with logarithmic discount.
- Gold labels are derived from OpenI MeSH terms, Problems fields, and non-negated text mentions.
